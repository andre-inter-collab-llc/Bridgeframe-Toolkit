# CancerSurv Sample Data

This directory contains sample datasets for the CancerSurv case study used throughout the Bridgeframe Toolkit.

## Files

- `sample_cases.csv`: Simulated cancer case records (de-identified)
- `facilities.csv`: Hospital and laboratory reference data
- `data_dictionary.md`: Field definitions and coding standards

## Usage

These datasets are for educational purposes only. They contain simulated data and should not be used for actual epidemiological analysis.

## Data Standards

Sample data follows NAACCR data standards where applicable:
- ICD-O-3 for histology and site coding
- TNM staging notation
- Standard demographic fields

## Privacy Note

All patient identifiers in sample data are entirely fictional. No real patient data is included.
